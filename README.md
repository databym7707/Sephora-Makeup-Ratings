# Sephora-Makeup-Ratings Data Analysis
## Project Overview
This project analyzes Sephora product review data to identify patterns in product performance, customer ratings, and review volume. The goal is to explore how review count and average rating relate to overall product success and customer perception.Analyzing product level performance using ratings and review volume from Sephora.

This project was created for portfolio purposes. Feedback, suggestions, and critiques are welcome.

---

## Tools Used
- SQL (SQLite)
- Google Sheets
- Tableau

---

## Dataset Information
Dataset sourced from a public Sephora dataset on Kaggle (uploaded by Nady Inky).  
The data was originally collected via a Python web scraper in March 2023.

The repository includes:
- The original dataset
- A cleaned & edited version of the dataset

---

## Data Cleaning Process
The original datasat was cleaned & prepared for analysis by:
- Removing unnecessary columns
- Standardizing column names
- Checking for and handling missing values
- Saving cleaned version in a separate tab for analysis
